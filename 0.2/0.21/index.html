Matrix Multiplication

Definition :

The product AB of two matrices A and B -- where A is a m by n matrix, and B is a n by l matrix -- is a m by l matrix in which the element which lies in the ith row and jth column is obtained by multiplying each element of the ith row of matrix A by the corresponding element of the jth column of matrix B and adding the results.
